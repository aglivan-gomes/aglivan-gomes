# <div align="center">Olá, sou Aglivan Gomes</div>  
  

Graduando em Design Digital na Universidade Federal do Ceará – Campus Quixadá.

Minha área principal é o design, sobretudo a área de modelagem, mas não consigo fugir de meu passado como programador. Possuo habilidades em desenvolvimento web com ênfase em backend, buscando criar soluções estruturadas, escaláveis e seguras para aplicações.

Em um cenário em constante evolução, acredito que o backend não é apenas a “engrenagem oculta” dos sistemas, mas o que garante confiabilidade e performance para que a experiência do usuário seja realmente fluida. Para mim, desenvolver é conectar tecnologia e pessoas de maneira sólida, eficiente e transparente.

  
  


## 🛠 Minhas habilidades

- **Linguagens:** JavaScript, TypeScript, Java, C#, C++ (básico), Kotlin (básico)
- **Front-end:** React, React Native
- **Backend / APIs:** Node.js, Express.js, NestJS (em estudo)
- **Bancos de Dados:** PostgreSQL, MongoDB, Firebase (Realtime Database/Firestore)
- **Ferramentas:** Git/GitHub, Docker (em aprendizado), Prisma ORM
- **Outros:** Integração com APIs, autenticação, deploy em serviços cloud
